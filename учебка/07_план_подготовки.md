products = products.OrderBy(p => p.Count).ToList();

// 3. Привязать к ListBox
ProductList.ItemsSource = products;

// 4. Получить выбранный элемент
if (ProductList.SelectedItem is Product p) { ... }

// 5. Добавить запись
db.Products.Add(newProduct);

### Знать структуру XAML:

```xml
<!-- Window → Grid → элементы -->
<Window ...>
    <Grid>
        <Grid.RowDefinitions>
            <RowDefinition Height="50"/>
            <RowDefinition Height="*"/>
        </Grid.RowDefinitions>

        <TextBlock Grid.Row="0" Text="Заголовок"/>
        
        <ListBox Grid.Row="1" x:Name="ProductList">
            <ListBox.ItemTemplate>
                <DataTemplate>
                    <StackPanel>
                        <TextBlock Text="{Binding Descrip}"/>
                        <TextBlock Text="{Binding Price}"/>
                    </StackPanel>
                </DataTemplate>
            </ListBox.ItemTemplate>
        </ListBox>
    </Grid>
</Window>
```

## Что НЕ нужно писать руками

- `ShoeshopContext.cs` — генерируется scaffold командой
- Все Entity классы (`Product.cs`, `User.cs` и т.д.) — тоже scaffold
- Вычисляемые свойства (`FinalPrice`, `HasDiscount`) — дописываются в `partial class`

---

## Команда scaffold (скопируй, не забудь)

В Package Manager Console (Tools → NuGet Package Manager → Package Manager Console):

```
Scaffold-DbContext "server=localhost;database=shoeshop;user=root;password=1234" Pomelo.EntityFrameworkCore.MySql -OutputDir Entities -ContextDir DbContexts -Context ShoeshopContext -Force
```
